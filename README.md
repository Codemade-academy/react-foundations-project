# React Foundations Project

##  Project Overview
This project demonstrates core React fundamentals including components, props, state, and routing.  
The application simulates a simple product showcase platform.

---

##  Objective
To build a multi-page React application using functional components and React Router.

---

##  Features
- Multi-page navigation
- Product listing page
- Product detail page
- Cart counter
- Responsive design

---

##  Technologies Used
- React (Vite)
- React Router DOM
- CSS / Tailwind
- JavaScript (ES6+)

---

##  Folder Structure

- src/
- components/
- pages/
- assets/
- App.jsx
- main.jsx
